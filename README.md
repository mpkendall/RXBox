# RXBox Zero
### an all-in-one receive station for everything radio

The goal of this project is to sucessfully integrate an SBC (single-board computer), SDR (software-defined radio), touchscreen display, and a lithium battery powerpath into a single cohesive unit. It will be able to support user-built apps using an easy-to-use API for internal functions. The primary use case is for high-altitude ballooning, which demands real-time radio receive capabilities for trackers like [Tiny4FSK](https://github.com/New-England-Weather-Balloon-Society/Tiny4FSK).

Development progress can be followed in the [JOURNAL.md](/JOURNAL.md) file!
